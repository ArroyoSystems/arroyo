# Documentation specs

This folder contains more focused architecture specs that complement the root `ARCHITECTURE.md`.

- `control-plane-spec.md`: API, controller, compiler, node, admin surfaces, and scheduling modes.
- `planning-and-execution-spec.md`: SQL planning, logical program lowering, worker execution, state, checkpoints, connectors, and UDFs.
- `deployment-spec.md`: local, Docker, and Kubernetes deployment models plus storage and configuration boundaries.
- `crate-map.md`: workspace-level inventory of Rust crates and their responsibilities.
